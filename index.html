Последние реализованные проекты:

Изготовление мобильного офиса из 2-х контейнеров для компании

Задача: Переоборудовать 2 контейнера в мобильный офис на 4 рабочих местах.

Необходимо: Четыре рабочих места, оборудованная зона отдыха и место для приема пищи.

Изготовление мобильного офиса из 1-го контейнера для компании

Задача: Переоборудовать один контейнер в мобильный офис на два рабочих места.

Необходимо: Установить два рабочих места, зону для приема пищи и сан. узел. Офис оборудовать всей необходимой для работы мебелью.

Изготовление строительной бытовки из 1-го контейнера для компании

Задача: Переоборудовать 1 контейнер под строительную бытовку для установки на строительной площадке.

Необходимо: Расположить душевую кабинку для сотрудников, установить поворотно-откидное окно с москитной сеткой и закрепить 4 шкафчика для переодивания.

Изготовление поста охраны для супермаркета

Задача: Изготовить пост охраны размером 2*2 м для размещения на парковке супермаркета в г. Одесса.

Необходимо: Разместить одного сотрудника, который будет работать посменно (24 часа).
